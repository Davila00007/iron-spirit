# 🏋️ Iron Spirit — Sitio Web de Gimnasio

Proyecto de sitio web completo para un gimnasio, desarrollado con **HTML, CSS y JavaScript puro** como parte de mi aprendizaje en desarrollo web.

## 📸 Vista general

Sitio web moderno, oscuro y responsive para el gimnasio **Iron Spirit**, con diseño profesional, animaciones y secciones interactivas.

---

## 🛠️ Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura y contenido |
| CSS3 | Estilos, animaciones y responsive design |
| JavaScript (Vanilla) | Interactividad y funcionalidades dinámicas |
| Font Awesome 6 | Íconos |
| Google Fonts | Tipografía (Montserrat) |


## 📂 Estructura del proyecto

```
iron-spirit/
├── index.html        # Estructura principal del sitio
├── style.css         # Todos los estilos
├── javascript.js     # Lógica e interactividad
├── README.md         # Documentación del proyecto
└── img/              # Imágenes del sitio
```

---

## 📋 Secciones del sitio

- **Header** — Barra de navegación fija con efecto glassmorphism y menú hamburguesa responsive
- **Home** — Sección hero con efecto parallax en el título al mover el mouse
- **Instalaciones (Banner)** — Banner full-width con imagen de fondo y overlay animado
- **Galería (Carrusel)** — Carrusel automático con 4 slides, barra de progreso y swipe táctil
- **Services** — 6 tarjetas de servicios con imagen de fondo y efecto hover en rojo
- **Productos** — Tienda con 9 productos filtrados por categoría (Suplementos, Ropa, Equipos)
- **Pricing** — 3 planes de membresía con toggle de precio mensual/anual
- **Contact** — Formulario de contacto + datos del gym + redes sociales + footer

## ✨ Funcionalidades destacadas

- ✅ Diseño **100% responsive** (mobile, tablet, desktop)

## ⚙️ Cómo usar el proyecto

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/iron-spirit.git
```

2. Abre la carpeta del proyecto en tu editor (VS Code recomendado)

3. Abre el archivo `index.html` en tu navegador o usa la extensión **Live Server** en VS Code

> No requiere instalación de dependencias ni servidor especial.

---

## 🎨 Personalización

Para adaptar el sitio a tu gym, edita estas variables en `style.css`:

```css
:root {
    --color-main: #ff0800;  /* Color principal (rojo) */
    --bg-main: #141418;     /* Fondo oscuro */
    --white: white;
    --tarjetas: #0000007a;  /* Fondo de tarjetas */
}
```
